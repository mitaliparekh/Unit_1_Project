Key Concepts



In this section, we will discuss five key concepts.


    Data vs. Information

    Data is raw information that's gathered from various sources such as user input (Morley, 2014).

    Information is the gathered data transformed into a presentable form for users to understand (Morley, 2014).

    Hardware

    Hardware is the physical computer and its components such as the motherboard (Morley, 2014).

    Embedded Computers

    Embedded computers are small computers that are inserted into products such as washing machines (Morley, 2014).

    Server

    A server holds all the data for a user and/or a company. They have higher storage capacity compared to a desktop computer (Morley, 2014).

    Accessing a Network or the Internet

    To access the network, one can use the computer's adaptor such as WIFI along with a username and password generated by the network provider. To access the internet or a webpage on the internet, one can use a URL (Morley, 2014).
